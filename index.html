Dashboard Sector Minero Energético Colombia

Panel de control interactivo para el análisis estratégico de proyectos energéticos y mineros. Visualiza datos, optimiza recursos y toma decisiones basadas en evidencia.

Streamlit Python Pandas Plotly SQLite
Ver en GitHub Quick Start

Desarrollado por Feibert Guzmán

Módulos del Dashboard

Funcionalidades clave para el análisis integral del sector.

Inicio

KPIs generales, distribución energética y fuentes de inversión en tiempo real.

Proyectos

Listado detallado y filtrado de proyectos por ubicación geográfica y tipo de energía.

Eficiencia

Series temporales de generación, costos operativos e indicadores económicos clave.

Inversiones

Análisis profundo de montos de capital y fuentes de financiamiento (Gobierno, Privado, ONG).

Empresas

Mapeo de actores del ecosistema y su participación porcentual por proyecto.

Comunidades

Impacto social y beneficiarios locales asociados a cada proyecto energético.

Stack Tecnológico

Frontend & Visualización

Streamlit para la interfaz rápida y Plotly Express para gráficos interactivos de alta calidad.

Backend & Datos

Python 3.8+ con SQLite3 para gestión ligera de bases de datos relacionales.

Data Engineering

Pandas para procesos ETL, limpieza de datos y análisis exploratorio robusto.

Esquema de Datos

📦 SectorMineroEnergeticoColombia.db
  • ├── tipos_energia # Catálogo: Solar, Eólica, H2, Biomasa
  • ├── proyectos # 5 proyectos piloto con metadata geoespacial
  • ├── eficiencia_energetica # 50 registros: kWh, costos, KPIs
  • ├── inversiones # Fuentes: Gobierno, Privado, ONG
  • ├── empresas # Operadores por proyecto
  • └── comunidades_energeticas # Beneficiarios locales

Notas Técnicas

  • Datos sanitizados en init_db.py (corrección de formatos y typos).
  • Conexión BD optimizada con @st.cache_data.
  • Diseño Mobile-First totalmente responsive.

🚀 Inicio Rápido

Sigue estos pasos para ejecutar el dashboard localmente.

1. Clonar repositorio

bash 
git clone https://github.com/FeibertGuzman/streamlit2
cd streamlit2

2. Instalar dependencias

bash 
pip install -r requirements.txt

3. Inicializar base de datos

bash 
python init_db.py

4. Ejecutar aplicación

bash 
streamlit run app.py

La app estará disponible en: http://localhost:8501
